Stocker

A Python wrapper for the Yahoo Finance API.

stocker on github
Import
import stocker
Get quotes (all properties)
stocker.quotes( ['GOOG', 'AMZN'] )
Get quotes (specific properties)
stocker.quotes( ['GOOG'], ['pe_ratio', 'eps', 'bid'] )
Get stock history
stocker.history( ['AAPL', 'GOOG'], "02/10/2008", "04/24/2012", 'w' )
Get history for AAPL and GOOG from 02/10/2008 to 04/24/2012 on a weekly interval.
Interval can also be 'd' (daily), 'm' (monthly), and 'v' (dividends).
Get data for all sectors
stocker.sectors()
Get data for industries
stocker.industries( ['healthcare', 'technology'] )
Get company data by industry
stock.companies( ['gold', 'personal_computers'] )
View available quote properties, industries and companies
$ cat properties.json
